Why another Java source code beautifier?

BeautyJ had started as a pure exercise for the author's study. It originates from experiments with the JavaCC compiler compiler and with UML based coding techniques using ArgoUML and Poseidon for UML. Almost by itself it grew to become a source code transformation tool with open API interfaces. In combination with the development of the AMODA – Abstract Modeled Application by the same author, it serves as a full size demonstration of the AMODA concepts.
